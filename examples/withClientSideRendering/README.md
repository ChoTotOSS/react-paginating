# Example with Client Side Rendering

## How to use

Download the example
[or clone the repo](https://github.com/ChoTotOSS/react-paginating):

```bash
cd examples/withClientSideRendering
```

## Development

Install it and run:

npm

```bash
npm install
npm run dev
```

yarn

```bash
yarn install
yarn dev
```

## Production

Install it and run:

npm

```bash
npm install
npm run prod
```

yarn

```bash
yarn install
yarn prod
```
