# Example with static data

## How to use

Download the example
[or clone the repo](https://github.com/ChoTotOSS/react-paginating):

```bash
cd examples/withBasic
```

## Development

Install it and run:

npm

```bash
npm i
npm start
```

yarn

```bash
yarn install
yarn start
```

## Production

Install it and run:

npm

```bash
npm install
npm start:prod
```

yarn

```bash
yarn install
yarn start:prod
```
